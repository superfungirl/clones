Start Elld (with a local account)

To start Elld, use the following command:
elld start -a "0.0.0.0: 9000" --account "eFjwxLSvseLSafgqMk4UGsnBNUfUy6N2hh"
Show Response
elld account create # Output # New account created, encrypted and stored # Address: eFjwxLSvseLSafgqMk4UGsnBNUfUy6N2hh
Show Response
To start Elld, use the following command:
elld start \ - a "0.0.0.0:9000" \ - j "ellcrys: //" 12D3KooWGZxHXA3P9SxyMRYecgyKggAmdviTkKKVnmUCaWU
bbQoJ@35.241.79.20:9000"
Show Response
INSTRUCTION When the node starts, an ephemeral key is used to derive the node's identity when interacting with other nodes on the network. Ephemeral keys are temporary; They are disposed of when the node stops running.
